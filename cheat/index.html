My Sheet of Cheats

Listen, there are a million cheat sheets out there. Some are great, some have only one line that you want. Everyone should make their own, and these are mine. Maybe they'll help you.

Windows

The Net Command

  • Net accounts – list the local password policy. Note that I said LOCAL – this doesn’t work on a Domain Controller.
  • Net config server|workstation – provides basic settings about the server or workstation such as OS version and domain name.
  • Net start – lists the services that are running. Note that this is not the processes that are running, but the Windows services that are running
  • Net localgroup administrators – fast way to list the accounts that are local admin
  • Net stop X – stop the X service
  • Net pause X – pause the X service
  • Net continue X – continue the X service
  • Net session – lists active session with other hosts on the network
  • Net view – list other devices on the network
  • Net user – list the local users
